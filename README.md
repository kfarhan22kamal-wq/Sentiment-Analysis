# Sentiment Analysis of Amazon Reviews

This project predicts whether an Amazon review is positive or negative using machine learning.

## Features
- Data preprocessing
- TF-IDF vectorization
- Logistic Regression model
- VADER comparison
- Evaluation metrics and plots

## Project Structure
- data/
- notebooks/
- src/
- outputs/

## How to Run
```bash
pip install -r requirements.txt
python run_all.py
```

## Challenges
- Cleaning noisy review text.
- Choosing a suitable text vectorizer.
- Comparing ML predictions with VADER.
